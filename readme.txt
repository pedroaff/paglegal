== Description ==

### Add PagLegal gateway to WooCommerce ###

This plugin adds PagLegal gateway to WooCommerce.

Please notice that WooCommerce must be installed and active.

Requires WordPress 4.9, WooCommerce 3.2 and PHP 7.0.


== Installation ==

Upload the PagLegal plugin to your WooCommerce site, activate it and you're done!
